Was macht intraRP aus?

Open-Source

intraRP ist ein Open-Source-Projekt, das von der Community unterstützt wird.

EASY-PEASY

Das ganze System ist einfach aufzusetzen und zu konfigurieren.

Ingame

Mittels intraTab ist auch eine Verbindung in FiveM möglich.

intraTab - Dein Ingame Notfallprotokoll

intraTab ist eine Ingame-Anwendung, die es dir ermöglicht, intraRP direkt im Spiel zu nutzen. So kannst du Rettungsdiensteinsätze dokumentieren ohne das Spiel verlassen zu müssen.

Interaktive An-/Abmeldung

eNOTF bietet die Möglichkeit zur direkten Anmeldung auf ein Fahrzeug. Hinterlegte Daten werden automatisch im Protokoll übernommen.

Einsatzverwaltung

Angemeldete Fahrzeuge sehen eine Übersicht ihrer offenen/angelegten Einsätze.

Schnellzugriff

In der Übersicht bieten sich Schnellzugriff-Knöpfe, mit diesen können mitunter die Fahrzeug-Checklisten angezeigt werden.
